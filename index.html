IPL Data API Documentation

API endpoints for accessing Indian Premier League cricket statistics.

GET /
API welcome message
GET /sid/all
Get all matches data
GET /sid/teams
Get all teams that have ever played in IPL
GET /sid/team_vs_team_info/{team1},{team2}
Get head-to-head information between two teams
Parameters: team1 (string), team2 (string)
GET /sid/team_vs_team_stats/{team1},{team2}
Get detailed statistics between two teams
Parameters: team1 (string), team2 (string)
GET /sid/team_vs_all/{team}
Get a team's performance against all opponents
Parameters: team (string)
GET /sid/batsman_record/{batsman}
Get batting statistics for a specific player
Parameters: batsman (string)
GET /sid/bowler_record/{bowler}
Get bowling statistics for a specific player
Parameters: bowler (string)
GET /sid/batsman_vs_bowler/{batsman},{bowler}
Get performance data for a batsman against a specific bowler
Parameters: batsman (string), bowler (string)
GET /sid/batsman_record/{team}
Get venue-wise performance for a team (Note: Endpoint path appears mislabeled as 'batsman_record')
Parameters: team (string)
